# SRTM Elevationa and Slope Data

The 500m SRTM elevation data was originally downloaded from CGIAR CSI but is no longer directly available. Other SRTM products (e.g., 90m) are available online. Slope was calculated from the 500m SRTM data using GDAL's `gdaldem slope` command within QGIS. The slope is in degrees.

See:
https://www.usgs.gov/centers/eros/science/usgs-eros-archive-digital-elevation-shuttle-radar-topography-mission-srtm-1
https://srtm.csi.cgiar.org/srtmdata/
