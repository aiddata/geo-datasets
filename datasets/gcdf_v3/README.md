## China Project Boundaries and Datasets

- `gcdf_v3_dynamic/` is a Prefect flow (see its own directory) that downloads
  AidData's GCDF v3.0.1 project geometries and prepares the filterable
  feature dataset used by GeoQuery's dynamic query tooling.
- `gcdf_v3_boundaries/` and `gcdf_v3_static/` are generated by a separate
  repo, not by a flow in this directory:
  - See https://github.com/aiddata/china-osm-geodata/tree/master/examples/generate_buffers for code to generate boundary buffers
  - See https://github.com/aiddata/china-osm-geodata/tree/master/examples/rasterize for code to generate raster datasets
