# Global Solar Atlas: PV Output

Long-term yearly average of daily totals of potential photovoltaic
electricity production (PVOUT, kWh/kWp), covering the period from
1994/1999/2007 (depending on the region) to 2015, from the Global Solar
Atlas (World Bank Group / Solargis).

## Manual download

There is no programmatic/bulk download endpoint for this dataset — it's
downloaded by hand from the web tool:

1. Go to https://globalsolaratlas.info/download/world
2. Under **World**, find **PVOUT - Photovoltaic power potential** and the **Annual daily totals** layer (PVOUT - LTAy_AvgDailyTotals)
3. Download the GeoTIFF
