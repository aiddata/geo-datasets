# Protected Areas (IUCN Categories) - June 2024

The World Database on Protected Areas (WDPA) is the most comprehensive global database of marine and terrestrial protected areas. Dataset downloaded June 2024. This is a categorical raster indicating the IUCN category (or relevant information when no category exists) of the protected area covering each pixel.

## Quick start

1. Review and edit the variables in `config.toml` as needed
    - `download_url`
    - `raw_dir` is a working/output directory
    - `output_dir` is a working/output directory
    - `max_retries`
    - `overwrite_download`, if true, overwrites existing files rather than skipping
    - `overwrite_output`, if true, overwrites existing files rather than skipping

## Source

IUCN and UNEP-WCMC
